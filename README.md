# SnapLeaf

SnapLeaf 是一款 macOS 菜单栏截图与全局标注工具。专为设计师、产品和开发者打造：快捷键一键截图后，直接进入无边框标注白板，支持键盘输入法打字、箭头、矩形、画笔等标注，并导出 AI 友好的图片 + JSON 元数据。

## 核心功能

- **全局截图**：区域 / 全屏 / 整页 / 窗口 / 手动滚动长图。
- **即时标注白板**：截图后直接进入无边框标注层，支持滚轮缩放、空格拖拽平移、撤销重做。
- **键盘文字标注**：在截图上调用输入法直接打字，支持中/英文、多行文本、颜色与字号调整。
- **丰富标注工具**：矩形、箭头、画笔、马赛克、高亮、序号，配合颜色、粗细选择与磁吸辅助线。
- **贴图置顶**：将截图钉在屏幕最上层，支持透明度、缩放、保存、复制与 Cmd+W 关闭。
- **OCR 识别**：一键提取截图中的中英文字。
- **历史管理器**：缩略图网格浏览，双击重新标注或置顶。
- **AI 友好导出**：⌘⇧C 复制图片 + JSON 元数据，⌘S 保存 PNG + JSON 文件。
- **快捷键自定义**：在菜单栏 > 设置中按需调整。

## 默认快捷键

| 操作 | 快捷键 |
|------|--------|
| 全局 / 全屏截图 | `⌘Q` |
| 局部截图 | `⌘W` |
| 整页截图 | `⌘R` |
| 窗口截图 | `⌘E` |
| 截长图（滚轮/触控板手动选取） | `⌘T` |
| 标注上次截图 | `⌘A` |
| 退出应用 | `⌘⇧Q` |
| 复制图片 + JSON | `⌘⇧C` |
| 保存 PNG + JSON | `⌘S` |

## 标注白板操作

- 滚轮缩放画布
- 按住空格拖拽平移
- 多步撤销 / 重做
- 双击文字框编辑
- 磁吸辅助线对齐元素

## 构建

```bash
./build.sh
```

构建产物为 `SnapLeaf.app` 与 `SnapLeaf.zip`，已做 ad-hoc 签名，可直接分发。
