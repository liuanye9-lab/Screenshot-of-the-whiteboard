macOS 上极简的截图 + 全局标注白板。截图后用键盘输入法直接打字标注，配合箭头、矩形、画笔，让 AI 一眼读懂你的修改意图。

核心功能

为设计师、产品与开发者打造的轻量截图工作流。

全局截图 + 即时标注

快捷键唤起区域/全屏/窗口/长图截图，截图后立即进入无边框标注白板，支持缩放、平移与撤销重做。

键盘文字标注

在截图上直接调用输入法打字，支持中/英文、多行文本、颜色与字号调整，像聊天一样把修改意见写在图上。

丰富标注工具

矩形、箭头、画笔、马赛克、高亮、序号等工具，配合颜色、粗细选择与磁吸辅助线，精确定位修改位置。

贴图置顶

将截图钉在屏幕最上层，支持透明度调节、滚轮缩放、双击适配/原始尺寸、右键保存与 Cmd+W 关闭。

OCR 识别 + 历史管理

一键提取截图中的中英文字，历史浏览器以缩略图网格展示，双击即可重新标注或置顶。

AI 友好导出

标注结果一键复制图片 + JSON 元数据，或保存 PNG + JSON 文件，让 AI 准确理解修改意图。

使用指南

四步上手，从截图到标注一气呵成。

下载安装

点击上方「下载 SnapLeaf」获取 SnapLeaf.zip，解压后将 SnapLeaf.app 拖入「应用程序」文件夹，双击启动。首次运行需在系统设置中授权「屏幕录制」和「辅助功能」权限。

快捷截图

菜单栏会出现相机图标。按 ⌘W 局部截图（框选区域），⌘Q 全屏截图，⌘E 窗口截图，⌘T 长图截图。所有快捷键均可在设置中自定义。

标注白板

截图后自动进入标注白板。按 ⌘3 选文字工具，在图上单击即可用键盘输入法打字标注。还可使用矩形 ⌘1、箭头 ⌘2、画笔 ⌘4、马赛克 ⌘5、高亮 ⌘6、序号 ⌘7 等工具。滚轮缩放，空格拖拽。

导出与 AI 协作

按 ↵ 完成标注，自动复制到剪贴板。按 ⌘⇧C 复制图片 + JSON 元数据（包含每个标注的位置、颜色、文字），直接粘贴给 AI 助手，让 AI 精准理解你的修改意图。也可 ⌘S 保存 PNG + JSON 文件。
